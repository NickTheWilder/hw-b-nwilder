## Execution
Using Node Packet Manager @ [link](https://www.npmjs.com/)
Using a json-server to create a mock database to simulate data @ [link](https://www.npmjs.com/package/json-server)

Script has been created to run the server-side of the application
- default port is localhost:5000
- located in packages.json if port is needed to be changed

To start the front-end of React
```bash
npm start
```

Open a new console -> execute the following bash to cmd to start the json-server
```bash
npm run server
```

## Notes
This application is a ReactJs Application, using Axios to access a JSON database.
